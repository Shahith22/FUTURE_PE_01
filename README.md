# AI Website Copy Generator

## 📌 Project Overview
This project uses prompt engineering to generate high-quality website copy for local businesses.

## 🏢 Business Chosen
PromptCraft Studio (Freelancing AI & Web Agency)

## 🎯 Objective
To create structured prompts that generate:
- Homepage copy
- Services content
- Call-to-action sections

## ⚙️ Tools Used
- ChatGPT

## 🧠 Prompt Strategy
Prompts were designed to:
- Include business context
- Specify tone and audience
- Generate structured outputs

## 📂 Project Structure
- prompts/ → contains reusable prompts
- outputs/ → contains generated content

## 🚀 Outcome
Generated website copy that is:
- Clear and persuasive
- Conversion-focused
- Ready for real-world use
